# Papers

Every paper read gets an entry here, regardless of how thoroughly it was read — a skim gets a short entry, a paper you reproduce gets a long one. Status tracks depth of engagement, not quality.

ID format: `PAPER-001`. See [naming convention](../docs/naming-convention.md).

To add a new paper, copy [`_template.md`](_template.md), rename it `PAPER-XXX-short-title.md`, and fill it in.
