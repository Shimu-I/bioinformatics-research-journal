# Weekly Journal

The narrative spine of the repository. Every other folder holds a discrete fact; this folder holds the story that connects them week by week. Write it even in weeks where little happened — a short "quiet week, mostly reading" entry keeps the timeline unbroken.

ID format: `WEEK-2026-W03` (ISO week number). Entries are grouped by year subfolder: `journal/2026/`. See [naming convention](../docs/naming-convention.md).

To add a new week, copy [`_template.md`](_template.md) into the correct year folder, rename it `WEEK-YYYY-Www.md`, and fill it in as the week goes on.
