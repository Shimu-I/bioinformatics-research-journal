# Glossary

New terminology, one entry per term, in your own words. The point is not a dictionary definition — it's proof that you understood the term well enough to explain it, plus a record of where you first ran into it.

ID format: `TERM-001`. See [naming convention](../docs/naming-convention.md).

To add a new term, copy [`_template.md`](_template.md), rename it `TERM-XXX-term-name.md`, and fill it in.
