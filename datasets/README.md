# Datasets

Documentation for every dataset used or explored — including ones that turned out not to be useful. Note quirks and gotchas here so you don't rediscover them the hard way a second time.

ID format: `DATA-001`. See [naming convention](../docs/naming-convention.md).

To add a new dataset, copy [`_template.md`](_template.md), rename it `DATA-XXX-short-title.md`, and fill it in.
