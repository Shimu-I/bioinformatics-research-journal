# Reproductions

Attempts to reproduce a result from a paper. An entry here should exist even for failed or partial attempts — those are often more informative than successful ones.

ID format: `REPRO-001`. See [naming convention](../docs/naming-convention.md).

To add a new reproduction, copy [`_template.md`](_template.md), rename it `REPRO-XXX-short-title.md`, and fill it in.
