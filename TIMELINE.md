# Timeline

A single chronological index of everything that happens in this research journal. Each entry is one line and links to the full record. This file answers "when did X happen." For narrative and reasoning, see [journal/](journal/README.md).

## How to Add an Entry

Whenever you create or close out an item anywhere in the repository, append one row here, in date order:

```
| YYYY-MM-DD | Type | ID | Title | Link |
```

`Type` is one of: `Question`, `Paper`, `Reproduction`, `Experiment`, `Decision`, `Mistake`, `Principle`, `Meeting`, `Dataset`, `Term`, `Idea`, `Review`, `Week`.

This is the only file where every item in the repository is guaranteed to appear exactly once, so keep it up to date as you go rather than reconstructing it later.
