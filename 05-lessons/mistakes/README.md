# Mistakes

A specific, honest record of things that went wrong — technical errors, wrong assumptions, misread papers, wasted time. Log these without editorializing about how you feel about them; the value is in the root cause and the correction, not self-criticism.

ID format: `MIST-001`. See [naming convention](../../docs/naming-convention.md).

To add a new mistake, copy [`_template.md`](_template.md), rename it `MIST-XXX-short-title.md`, and fill it in.
