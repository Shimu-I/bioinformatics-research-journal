# Archived Inbox

Processed raw notes, kept for the record. Once an [`inbox/`](../../inbox/README.md) file has been extracted into permanent entries, it moves here — unedited.

## Rules

- Files land here exactly as they were when extracted. Do not rewrite, tidy up, or delete them afterward — see [GUIDE.md, Section 14](../../GUIDE.md#14-things-never-to-do). The extraction record (which files were updated, why, and what stayed temporary) is what makes an archived file traceable later; editing the source note after the fact breaks that trail.
- Naming stays the same as in `inbox/`: `YYYY-MM-DD-session-NN.md` or `YYYY-MM-DD-PAPER-XXX.md`.
- Archived inbox files are not part of the cross-linking system described in [docs/cross-linking-strategy.md](../../docs/cross-linking-strategy.md) — they don't need a `Related` section, and other entries don't link back to them. If something in an inbox file turned out to matter, it should already have been extracted into a real entry that *is* linked. Treat this folder as a raw-notes archive, not a navigable part of the knowledge base.
