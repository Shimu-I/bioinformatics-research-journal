# Questions

Research questions are the starting point of the lifecycle: `Question -> Paper -> Reproduction -> Experiment -> Decision -> Lesson / Principle`. A question doesn't need to be sophisticated to log here — "why does everyone use tool X for this" is a legitimate starting question.

ID format: `Q-001`. See [naming convention](../docs/naming-convention.md).

To add a new question, copy [`_template.md`](_template.md), rename it `Q-XXX-short-title.md`, and fill it in.
