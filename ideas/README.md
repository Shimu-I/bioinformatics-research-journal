# Ideas

A parking lot for ideas that aren't immediately actionable — usually because they need a stronger baseline, more background, or a clearer question first. Most ideas here won't be explored right away. That's the point: capture them now so they aren't lost, and revisit periodically (see monthly review, [reviews/](../reviews/README.md)).

ID format: `IDEA-001`. See [naming convention](../docs/naming-convention.md).

To add a new idea, copy [`_template.md`](_template.md), rename it `IDEA-XXX-short-title.md`, and fill it in.
