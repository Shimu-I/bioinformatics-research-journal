# Monthly Reviews

A monthly step back from weekly detail. Where the weekly journal ([journal/](../journal/README.md)) narrates what happened, a monthly review looks across several weeks for patterns — recurring mistakes, the biggest lessons, and whether the roadmap still makes sense.

ID format: `REVIEW-YYYY-MM`, e.g. `REVIEW-2026-07`.

To add a new review, first run the [Repository Audit](../docs/prompts/repository-audit.md) prompt over the repository, then copy [`_template.md`](_template.md), rename it `REVIEW-YYYY-MM.md`, and fill it in using the audit's findings alongside the month's journal entries.
