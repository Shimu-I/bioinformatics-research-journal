# Faculty Meetings

Notes from meetings with faculty, advisors, or mentors. Capture advice precisely enough that you can act on it weeks later without needing to remember the tone of the conversation.

ID format: `MEET-001`. See [naming convention](../docs/naming-convention.md).

To add a new meeting, copy [`_template.md`](_template.md), rename it `MEET-XXX-short-title.md`, and fill it in as soon as possible after the meeting.
