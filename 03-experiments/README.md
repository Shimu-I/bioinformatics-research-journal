# Experiments

Experiment logs — anything with a hypothesis, a setup, and a measurable result. This includes experiments run as part of a reproduction attempt, or standalone exploratory experiments.

ID format: `EXP-001`. See [naming convention](../docs/naming-convention.md).

To add a new experiment, copy [`_template.md`](_template.md), rename it `EXP-XXX-short-title.md`, and fill it in.
