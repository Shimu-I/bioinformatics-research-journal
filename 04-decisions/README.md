# Decisions

A log of research decisions and the reasoning behind them — tool choices, direction changes, scope cuts, anything where a future you might otherwise wonder "why did I do it this way." The point of this folder is to preserve the *why*, not just the *what*.

ID format: `DEC-001`. See [naming convention](../docs/naming-convention.md).

To add a new decision, copy [`_template.md`](_template.md), rename it `DEC-XXX-short-title.md`, fill it in, and add a one-line row to [WHY.md](../WHY.md) — the full reasoning lives here, the quick-recall version lives there.
