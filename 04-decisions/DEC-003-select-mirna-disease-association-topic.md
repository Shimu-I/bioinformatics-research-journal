# DEC-003 — Select miRNA–Disease Association Prediction as primary research topic

| Field | Value |
|---|---|
| ID | DEC-003 |
| Date | 2026-W29 |
| Status | Active |
| Superseded By | — |

## Context

Faculty suggested several active bioinformatics research areas: miRNA–Disease Association Prediction, Microbe–Disease Association, ncRNA–Disease Association, scRNA-seq, Drug Target Interaction (DDI), Spatial Transcriptomics, Multi-Omics Integration, Spatial Multi-Omics Integration, and scATAC-seq. See [MEET-002](../meetings/MEET-002-research-areas-and-paper-categories.md).

## Decision

Narrow focus to **miRNA–Disease Association Prediction** after exploring the suggested areas.

## Alternatives Considered

| Option | Pros | Cons |
|---|---|---|
| miRNA–Disease Association Prediction (chosen) | Well-defined prediction task, computationally tractable, papers in the area tend to have code and datasets | Requires validating that a specific paper is actually reproducible |
| Other suggested areas (scRNA-seq, Multi-Omics Integration, etc.) | Also active and relevant | Not yet explored in depth; deprioritized for now |

## Rationale

Consistent with [DEC-001](DEC-001-computational-ml-first-approach.md) — favors a computationally well-scoped problem — and with faculty advice to prioritize topics/papers with available source code, datasets, and reproducible experiments (see [PRIN-002](../05-lessons/principles/PRIN-002-prefer-reproducible-papers.md)).

## Consequences / Outcome

Led directly to selecting [PAPER-001](../01-papers/PAPER-001-mirna-disease-baseline-candidate.md) as a first candidate baseline, which was subsequently rejected — see [DEC-004](DEC-004-reject-paper-001.md).
